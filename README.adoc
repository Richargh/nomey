= Nomey

An Electron application with Vue and TypesSript

== Project Setup

=== Node

[source,bash]
----
# via nvm
nvm use
# or fnm
fnm use
----

=== Install

[source,bash]
----
$ npm install
----

=== Development

[source,bash]
----
$ npm run dev
----

=== Build

[source,bash]
----
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
----
